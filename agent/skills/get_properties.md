description: Use when the user wants to see, list, or retrieve their properties.
---

Use the `get_properties` tool when the user wants to view a list of their properties.

When the results come back, present them to the user in a clear, readable format showing each property's address, postcode, price, bedrooms, bathrooms, property type, status, and any other relevant information.

This tool does not require any input from the user - it will automatically find the current agent's properties.
